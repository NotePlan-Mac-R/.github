## Overview

NotePlan Advanced is a markdown-based planning application that combines daily journaling, task management, and a visual calendar into a single workspace built around how a day actually unfolds.

Daily notes provide a fresh markdown page for every day automatically, giving tasks, journal entries, and time-blocked plans a natural home tied to the date they belong to.

Task references link the same task across multiple notes and projects, so completing it in one place updates its status everywhere it appears without manual duplication.

Plugin support extends the core planning environment with community-built templates and automation scripts, adapting the application to specific planning methodologies like GTD or bullet journaling.

Calendar view toggles between day, week, and month layouts, giving a flexible overview of scheduled tasks and notes at whichever zoom level the current planning context calls for.

Recurring tasks and templates automate the setup of repeating daily or weekly routines, reducing the manual effort of recreating the same structure in a fresh daily note each time.

For people who plan their day through writing and want their tasks, notes, and calendar to live in one connected markdown workspace, NotePlan Advanced is the planning tool built around that integration.

## Performance

NotePlan balances performance with system resource usage intelligently. Operations are handled in the background, keeping your Mac responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series Macs, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
